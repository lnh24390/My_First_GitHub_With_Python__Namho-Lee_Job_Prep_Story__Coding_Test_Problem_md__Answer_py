# 이남호의 취업 준비 이야기
## - 취업 준비생의 이야기 -
### PART1 - 학력
한국복지대학교 **AI반도체융합과**를 입학 재학중 한경대와 통합이 되어서 한경국립대학교로 출범하여 한국복지대학교  
**AI반도체융합과**를 졸업한 이후 전공심화에 합격하여 **컴퓨터공학과**로 전공심화 과정에 들어가 졸업시험을 보아서  
**쿼트라인**을 넘겨 한경국립대학교로 졸업을했다.  
### PART2 - 취업 준비와 2회의 실패 그리고 다시 시작된 공부
취업을 준비하려고 여기 저기 알아보다가 취업에 두번 실패하여   
좀더 공부해야겠다고 생각하여 **한빛+**((구)한빛출판네트워크) 홈페이지에 들어가서  
**이것이 취업을위한 코딩 테스트다 With파이썬** 한권을 구입하였다.  
그리고 여기서 나의 첫번째 깃허브 프로젝트가 시작된다.
